## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML, CSS e JavaScript
- dJango e Python
- Git e GitHub
- PostgreSQL
- Bit.io  

<hr>

## Projeto

O projeto foi desenvolvido com intuito de resolver um problema de uma empresa chamada Neide Caitano. O site uma landpage amigavel ao usuário que possui o objetivo de passar a informação para o cliente previamente. Dentro do site tem: 
- Imagens de serviços realizado
- Quais serviços o salão oferece
- O horário de atendimento
- Link para agendamento direto com a Neide Caitano 
- Um pouco da história do salão
- Informações de Contato
- Informações integradas ao Banco de Dados postegres
